# BlueBay – Página estática (demo)
Este paquete contiene una **landing page estática** lista para publicar.

## Contenido
- `index.html` – Página principal
- `assets/style.css` – Estilos CSS
- Imágenes servidas desde Unsplash (enlaces externos) y mapa de Google Maps embebido.

## Cómo usar (rápido)
1. Descarga y descomprime el .zip.
2. Abre `index.html` en tu navegador para verla localmente.
3. Para publicarla gratis:
   - **Netlify**: arrastra la carpeta a https://app.netlify.com/drop
   - **Vercel**: crea un proyecto estático y sube la carpeta.
   - **GitHub Pages**: crea un repo, sube los archivos y habilita Pages.

## Personalización
- Cambia textos en `index.html`.
- Edita colores/tipografías en `assets/style.css`.
- Reemplaza enlaces de imágenes por fotos oficiales del hotel si las tienes.
- Actualiza teléfonos y correo en la sección **Contacto**.

> Este sitio es una DEMO educativa, no afiliada oficialmente a BlueBay Hotels.
